钢结构笔记

第一章 绪论

钢结构优点：

  • 强度高，塑形，韧性好
  • 钢结构重量轻
  • 材质均匀，和力学计算的假定符合

钢结构缺点：

  • 耐腐蚀性差
  • 耐热不耐火
  • 可能发生脆性破坏

钢结构设计方法：

  • 以概率理论为基础的极限状态设计方法

第二章 钢结构的材料

钢结构对材料的要求：

  • 1.较高的强度：较高的抗拉强度fu和屈服点fy。
  • 2.足够的变形能力：较好的塑形和韧性。
  • 3.良好的工艺性能：冷加工，热加工和可焊性。

钢材的破坏：

  • 塑性破坏：由于形变过大，超过了材料或者构件可能的应变能力而产生的。
  • 脆性破坏：脆性破坏前塑形变形很小，甚至没有塑形变形，计算应力可能小于钢材的屈服点fy。

钢材主要性能：

  • 设计中以下屈服点为依据
  • 抗拉强度fu成为材料的强度储备
  • G=E/2(1+Y) #y为泊松比
  • 冲击韧性是钢材抵抗冲击荷载的能力，是钢材划分质量等级的依据

可焊性：

  • 可焊性:可焊性是指钢材通过普通的焊接工艺便可以得到合格的焊缝的性能。

温度对于钢材强度的影响:

  • 温度对于钢材强度的影响的趋势是随着温度升高，强度降低，应变增大；反之，随着温度的降低，强度略有增加，随着塑形和韧性的降低而变脆。

应力集中：

  • 当发生应力集中时候，钢材会变脆。

为什么在计算时候不考虑钢材的应力集中？

  • 由于钢材的塑性较好，在一定程度上使应力进行重分配，应力分布严重不均匀的现象趋于平缓。

反复荷载：

  • 在反复的荷载作用下，钢材会发生疲劳破坏，疲劳破坏属于脆性破坏

钢材疲劳计算：

  • 容许应力幅法
  • 荷载采用标准值
  • 容许应力幅与钢种无关

第三章 钢结构的连接

钢结构连接的基本原则：

  • 钢结构的连接必须符合安全可靠，传力明确，构造简单，制造方便和节约钢材的原则，

连接方式有：

  • 焊接连接，铆钉连接，螺栓连接。其中，焊接连接是现代钢结构中最主要的连接方式

文本转音频并自动播放

这个项目可以批量将文本转为音频，并且支持一般播放器播放，这里我们使用了python的一个库来播放，主要是源于我自己做的一个语音聊天机器人

注意：使用过程中如果报错以下错，请重新安装python-magic-bin

It is better to raise an ImportError since we are importing magic module raise
ImportError('failed to find libmagic.  Check your installation')

代码：

# encoding=utf-8
import requests
import pygame
import time
import eyed3

# 使用pygame播放的话需要设置歌曲名为非中文，使用如网易云这类的播放器就不用管这个
path = 'music.mp3'

# 转为音频文件
def get_mp3():
    text = '''
    央视网消息：近期，中央政治局同志首次向党中央和习近平总书记书面述职，这是他们履新近半年来交的首份答卷。习近平认真审阅了中央政治局同志的述职报告，并就中央政治局同志履行职责、做好工作、改进作风提出重要要求。
    '''
    url = 'https://tts.baidu.com/text2audio?cuid=baiduid&lan=zh&ctp=1&pdt=311&spd=6&tex='+text
    find = requests.get(url)
    with open(path,'wb') as f:
        f.write(find.content)
        f.close()

# 播放音频
def player():
    pygame.mixer.init()
    print("播放音乐")
    pygame.mixer.music.load(path)
    xx = eyed3.load(path)
    music_time = xx.info.time_secs
    pygame.mixer.music.play()
    time.sleep(music_time)
    pygame.mixer.music.stop()

if __name__ == '__main__':
    get_mp3()
    player()

Flask课程笔记

第一课 Flask学习之网页计算器，主要学习内容是表单的提交

注意：

  • 1.css,js,img这些文件应该放在main.py目录下新建一个叫static的文件夹存放
  • 2.html文件应该放在main.py目录下新建一个叫templates的文件夹存放

导入管理静态css，js，img文件工具

from flask import url_for

渲染html视图

from flask import render_template

控制url跳转

from flask import redirect

python文件main.py

!/usr/bin/env python
# -*- coding: utf-8 -*-
from flask import Flask,url_for
from flask import render_template
from flask import request
from flask import redirect
app = Flask(__name__)


# 访问主页时跳转到add
@app.route('/',methods=['GET','POST'])
def index():
    return redirect(url_for('add'))


@app.route('/add',methods=['GET','POST'])
def add():
    print(request.method)
    if request.method == 'POST':
        a = request.form['adder-1']
        b = request.form['adder-2']
        massage = int(a) + int(b)
        url_for('static', filename='style.css')
        return render_template('index.html', massage=massage,a=a,b=b)
    return render_template('index.html')

if __name__ == '__main__':
    app.run()
