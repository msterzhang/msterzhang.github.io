文本转音频并自动播放

这个项目可以批量将文本转为音频，并且支持一般播放器播放，这里我们使用了python的一个库来播放，主要是源于我自己做的一个语音聊天机器人

注意：使用过程中如果报错以下错，请重新安装python-magic-bin

It is better to raise an ImportError since we are importing magic module raise
ImportError('failed to find libmagic.  Check your installation')

代码：

# encoding=utf-8
import requests
import pygame
import time
import eyed3

# 使用pygame播放的话需要设置歌曲名为非中文，使用如网易云这类的播放器就不用管这个
path = 'music.mp3'

# 转为音频文件
def get_mp3():
    text = '''
    央视网消息：近期，中央政治局同志首次向党中央和习近平总书记书面述职，这是他们履新近半年来交的首份答卷。习近平认真审阅了中央政治局同志的述职报告，并就中央政治局同志履行职责、做好工作、改进作风提出重要要求。
    '''
    url = 'https://tts.baidu.com/text2audio?cuid=baiduid&lan=zh&ctp=1&pdt=311&spd=6&tex='+text
    find = requests.get(url)
    with open(path,'wb') as f:
        f.write(find.content)
        f.close()

# 播放音频
def player():
    pygame.mixer.init()
    print("播放音乐")
    pygame.mixer.music.load(path)
    xx = eyed3.load(path)
    music_time = xx.info.time_secs
    pygame.mixer.music.play()
    time.sleep(music_time)
    pygame.mixer.music.stop()

if __name__ == '__main__':
    get_mp3()
    player()

import eyed3

audiofile = eyed3.load("song.mp3")
audiofile.tag.artist = u"Integrity"
audiofile.tag.album = u"Humanity Is The Devil"
audiofile.tag.album_artist = u"Integrity"

audiofile.tag.track_num = 2

audiofile.tag.save()

Flask课程笔记

第一课 Flask学习之网页计算器，主要学习内容是表单的提交

注意：

  • 1.css,js,img这些文件应该放在main.py目录下新建一个叫static的文件夹存放
  • 2.html文件应该放在main.py目录下新建一个叫templates的文件夹存放

导入管理静态css，js，img文件工具

from flask import url_for

渲染html视图

from flask import render_template

控制url跳转

from flask import redirect

python文件main.py

!/usr/bin/env python
# -*- coding: utf-8 -*-
from flask import Flask,url_for
from flask import render_template
from flask import request
from flask import redirect
app = Flask(__name__)


# 访问主页时跳转到add
@app.route('/',methods=['GET','POST'])
def index():
    return redirect(url_for('add'))


@app.route('/add',methods=['GET','POST'])
def add():
    print(request.method)
    if request.method == 'POST':
        a = request.form['adder-1']
        b = request.form['adder-2']
        massage = int(a) + int(b)
        url_for('static', filename='style.css')
        return render_template('index.html', massage=massage,a=a,b=b)
    return render_template('index.html')

if __name__ == '__main__':
    app.run()

            <input type="text" placeholder="adder-2" name="adder-2" value="{{b}}">=
            <input type="text" readonly="readonly" placeholder="result" name="result" value="{{massage}}">
            <input type="submit" value="计算" onclick="add()">
        </form>

CAD2014安装错误

win10安装CAD2014两个常见安装错误，17年的事转到18年解决啦

一.CAD重装无法激活解决办法

  • 1.打开文件夹C:\ProgramData\FLEXnet，将FLEXnet文件夹里的文件全部删除。
  • 2.重新激活就好啦。

二.CAD2014打开出现显示驱动程序文件hdi丢失怎么办呢

  • 1.打开注册表按“win+r”组合键打开运行，再输入“regedit”,确定。

  • 2.Ctrl + F 搜索 “CustomHeidiDriverPath” ，将键值删除。

  • 3.重启CAD，ok。

  • 4.如果还报错，重复第二步，可能没有删除完全。
