用中文写python代码

作为一个程序员，可能大部分人只知道只有易语言使用中文编程，其实不然，python语言也可以使用中文编程，话不多说，上图。

注意:请使用python3.x的版本实验

代码：

# encoding=utf-8
from inspect import getframeinfo, currentframe


class 我的类(object):

    def 呐喊吧(我自己):
        我自己.钱 = int(input('我有多少钱：'))

        if 我自己.钱 > 6:
            print('买西瓜')
        elif 我自己.钱 > 5:
            print('买橘子')
        else:
            print('买不起,回去搬砖')

我的变量 = 我的类()
我的变量.呐喊吧()

Flask课程笔记

第一课 Flask学习之网页计算器，主要学习内容是表单的提交

注意：

  • 1.css,js,img这些文件应该放在main.py目录下新建一个叫static的文件夹存放
  • 2.html文件应该放在main.py目录下新建一个叫templates的文件夹存放

导入管理静态css，js，img文件工具

from flask import url_for

渲染html视图

from flask import render_template

控制url跳转

from flask import redirect

python文件main.py

!/usr/bin/env python
# -*- coding: utf-8 -*-
from flask import Flask,url_for
from flask import render_template
from flask import request
from flask import redirect
app = Flask(__name__)


# 访问主页时跳转到add
@app.route('/',methods=['GET','POST'])
def index():
    return redirect(url_for('add'))


@app.route('/add',methods=['GET','POST'])
def add():
    print(request.method)
    if request.method == 'POST':
        a = request.form['adder-1']
        b = request.form['adder-2']
        massage = int(a) + int(b)
        url_for('static', filename='style.css')
        return render_template('index.html', massage=massage,a=a,b=b)
    return render_template('index.html')

if __name__ == '__main__':
    app.run()

HTML文件index.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <link rel="stylesheet" href="../static/style.css" />
</head>
<body>
    <h1>flask学习</h1>
    <h2>计算器</h2>
    <div align="conter" style="marg-top:80px;">
        <form name="form1" method="POST">
            <input type="text" placeholder='adder-1' name="adder-1" value="{{a}}">+
            <input type="text" placeholder="adder-2" name="adder-2" value="{{b}}">=
            <input type="text" readonly="readonly" placeholder="result" name="result" value="{{massage}}">
            <input type="submit" value="计算" onclick="add()">
        </form>
    </div>

</body>
</html>

CAD2014安装错误

win10安装CAD2014两个常见安装错误，17年的事转到18年解决啦

一.CAD重装无法激活解决办法

  • 1.打开文件夹C:\ProgramData\FLEXnet，将FLEXnet文件夹里的文件全部删除。
  • 2.重新激活就好啦。

二.CAD2014打开出现显示驱动程序文件hdi丢失怎么办呢

  • 1.打开注册表按“win+r”组合键打开运行，再输入“regedit”,确定。

  • 2.Ctrl + F 搜索 “CustomHeidiDriverPath” ，将键值删除。

  • 3.重启CAD，ok。

  • 4.如果还报错，重复第二步，可能没有删除完全。
