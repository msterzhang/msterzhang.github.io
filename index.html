土的可松性：

土的可松性：是指天然状态下的土经过挖掘以后，内部组织破坏，体积增大，以后虽然经过压实，仍不能恢复原来的体积。
土的可松程度可以用可松系数来表示：

简单Python爬虫

1.requests库的使用

  • 关于requests库的使用参见Requests官方文档。

导入相关的库

import requests
import re

def get_img():
    url = 'http://jandan.net/ooxx'
    html = requests.get(url).text
    find_img_list = re.findall('img src="(.*)",html')
    for img in find_img_list:
        print(img)

if __name__ == '__main__':
    get_img()
