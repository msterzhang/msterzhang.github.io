爬虫库requests-html

  • 一个集下载html，解析HTML，执行javascript文件的爬虫库requests-html，来自requests库作者kennethreitz的最新作品

强大功能：

  • 自动执行js文件
  • 支持css选择器
  • 支持xpath选择
  • 更多强大功能点这里requests-html

注意：这个库目前只支持python3.6

安装：

pip install requests-html

例子：

  • 这里使用煎蛋做教程，主要是因为他的图片链接是通过js动态加载的，望大家不要过分请求，以免给人家服务器造成过大的压力，可以帮忙点点广告

网站分析:

请对比这两张图，你会发现在网站源码html里面是完全没有图片链接的

执行js代码:

r.html.render()

代码：

#!/usr/bin/env python
# -*- coding: utf-8 -*-
import re
from requests_html import HTMLSession
session = HTMLSession()


def get_img():
    r = session.get('http://jandan.net/ooxx')
    r.html.render()
    html = r.html.html
    img_list = re.findall('img src="(.*?)" style="max-width: 480px; ',html)
    for img_url in img_list:
        print(img_url)


if __name__ == '__main__':
    get_img()
