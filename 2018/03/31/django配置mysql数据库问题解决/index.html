django配置mysql数据库问题解决

Django中Mysql的配置

第一步：

# https://docs.djangoproject.com/en/1.10/ref/settings/#databases

# 原有sqlite3数据库
'''
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': os.path.join(BASE_DIR, 'db.sqlite3'),
    }
}
'''
# 更改为Mysql数据库
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': '数据库名字',
        'USER': '账号',
        'PASSWORD': '密码',
        'HOST': '127.0.0.1',
        'PORT': '3306',
        'CHARSET':'utf8',
    }
}

第二步：

打开C:\Users\Administrator\AppData\Local\Programs\Python\Python36\Lib\site-packages\django\db\backends\mysql__init__.py,输入

import pymysql
pymysql.install_as_MySQLdb()

第三步：

打开C:\Users\Administrator\AppData\Local\Programs\Python\Python36\Lib\site-packages\django\db\backends\mysql\base.py，注释掉这面这段代码

version = Database.version_info
# 注释掉这面这段代码
'''if version < (1, 3, 3):
raise ImproperlyConfigured("mysqlclient 1.3.3 or newer is required; you have %s" % Database.__version__)
